Bing

Introduction

Microsoft's answer to Google is the Bing search engine. It is the currently third largest handling 5% of the global search share. It is a culmination of different search engines that Microsoft has developed. It started with MSN search in 1998, then Windows Live Search in 2006. In 2007, it became Live Search and then finally Microsoft, in an effort to create a new identity rebranded the name to Bing in 2009. Microsoft announced a 10-year deal with Yahoo! in that same year where Yahoo's search engine would be replaced by Bing. Microsoft would pay Yahoo 88% of its Ad revenue. Yahoo would maintain their branding but its search results would say "powered by Bing".

Webpage Characteristics

Perhaps the most noticeable difference from Google is its search home page. It is image centered including a background image that changes daily featuring mostly notable places. Scrolling down one will see a series of featured images including images that are related to the topic of the day. Bing, thanks to Microsoft has big social media integration. This includes Outlook, X-box, MSN, and Windows.

With the save search function one can pick and choose which search results to save and it will be added to a collection functioning as a search engine bookmark. This makes it easy for a user go back to some information he or she wants without trying to remember the website name or typing it in the search bar and searching for it in the results page.

The biggest draw for Bing is its Microsoft rewards perk. It rewards points to users based on how much they use the search function. Users can redeem these points in a shop which includes gift cards, sweepstake entries and donations to nonprofit organizations.
