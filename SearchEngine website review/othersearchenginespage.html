Other Search Engines

These search engines, although they aren't as well known or used as much as Google or Bing still have certain unique perks they contain to make them worthwile to use. Some is a dedicated image search platform, others have a question and answer format and others with different languages to cater to a certain ethnic group. Some of these websites are featured below in the list.

  • Yahoo
  • Ask.com
  • Wolfram|Alpha
  • CC Search
  • Internet Archive
  • Yandex.ru
  • Baidu
Yahoo

  • Yahoo is one of the oldest search engines in the business. Incorporated in 1995 it is older than Google. Once one of the largest internet companies it has declined and eventually bought out by Verizon Communications. Still, it is one of the most popular websites with its global rank at #9. Yahoo offers a lot more services than just web search. Its news and media is the #1 most visited platform in the internet. Others includes Yahoo Answers, email, video sharing, Yahoo directory and finance.

Ask.com

  • Ask.com, founded in 1995 by Garrett Gruenner and David Warthen is a question and answering focused website. The website is solely based on a community where a user can enter a question in and get another user to answer it in which it archives it. Since the search results are based solely on user activity the results may not be as current as the ones in Google or Bing.
Wolfram|Alpha

  • This service officially launched in 2009 is a question and answer website. Unlike other web search engines that provides a results list of the most relevant data that matches the search query Wolfram Alpha answers purely on computation on a query. For example, a user inputs an equation for a slope and the site will generate a graph for that slope. With Wolfram|Alpha Pro users can upload files for analysis, get visuals for presentations and access to various web apps.
CC Search

  • CC Search is an image based web search engine by CC licenses in which the new format came out of beta in 2019. It searches more than 300 million copyright-free images from works under the CC license. The results page is neatly animated while loading in. Each image in the results page has an info tab with the title, author, source and picture details provided. A user can also copy an HTML embed code if they wish to embed it to their website.
Internet Archive

  • Founded in 1996 by Brewster Kahle, Internet Archive is a search engine that provides free access to digitized content such as books, movies, videos and games. It also archives the web, saving the website at a certain date such that a user can use the Wayback Machine to search for a web page as is on that day. Users can also upload their own work. In 2019 thus far, Internet Archive has saved 330 billion web pages and around 32 million digitized works.
Yandex.ru

  • Yandex Search is the most popular search engine in Russia in which it is used by more than 53% of users in Russia. The main website is in the russian language but there is also an international english version of it. Yandex also provides other services much like Google with email, maps, Cloud Computing and news content. It has support services for developers, advertisers and investors.
Baidu

  • For the chinese internet Baidu is the main internet company. With an internet population of 731 million as of 2016 it has huge traffic. In 2019, it beat out Bing to become the world's second most popular web search engine although internationally it isn't as popular as it is in China. Baidu was founded in 2000 by Robin Li. It's name is translated literally as "hundreds of times" representing the unending search. Web services include Baidu Postbar, its chinese language search engine, Baidu Knows, the Ask.com equivalent and Baidu Encyclopedia. Other services include maps, images and video search. Baidu has also adopted a mobile platform in which nowadays 60% of its revenue comes from.
